Tags em HTML

    Separções de tópicos no texto.
  • Titulos
    Nomes de chamada para textos.
  • Imagens
    Figuras para anexar ao site,
  • Abreviações
    Resmuo das palavras.
  • Códigos fonte
    Fomatação para códigos.
  • Listas numeradas
    Listas com ordem de precedencia.
  • Listas com marcadores
    Listas nomalmente usada para lembretes.
